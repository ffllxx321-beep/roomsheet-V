## 📐 DWG 识别工作流

### 方式一：当前楼层
1. 用户在平面视图中打开插件
2. 选择链接的 DWG 图纸
3. 选择"当前楼层"模式
4. 插件调用 Revit API `NewRooms2()` 自动放置房间
5. 提取 DWG 文字并匹配到房间
6. 若闭合区域无房间，按区域中心自动创建房间
7. 自动填充房间名称

### 方式二：框选区域
1. 用户在平面视图中打开插件
2. 选择链接的 DWG 图纸
3. 选择"框选区域"模式
4. 用户在视图中框选识别范围
5. 插件识别框选区域内的闭合区域
6. 创建房间并填充名称

## 🚀 部署方式

### 开发环境
1. 安装 Visual Studio 2022 + .NET 8 SDK
2. 安装 Revit 2026
3. 打开 `RoomManager.csproj` 编译

### 推荐：一键安装（无需手动复制）
1. 下载/解压项目后，直接双击 `install.bat`
2. 脚本会自动查找 `bin\Release\net8.0-windows` 下的产物
3. 如果本机有 `dotnet`，脚本会自动尝试编译后安装
4. 安装到：`C:\ProgramData\Autodesk\Revit\Addins\2026\`
5. 若插件未显示，运行 `verify.bat` 做依赖检查

### 分发安装包（给非开发同事）
1. 双击 `package_installer.bat`
2. 生成 `release\RoomManager_Payload`（插件文件）和 `release\RoomManager_Payload.zip`
3. 若电脑已安装 Inno Setup 6，会自动生成安装向导 EXE（Setup）
4. 将 EXE 发给使用者，按向导安装即可（无 EXE 时可发 zip）

### 安装向导（EXE）环境检查
- 启动安装向导时会检查：
  1. 是否安装 `Revit 2026`
  2. 是否存在 `RevitAPI.dll / RevitAPIUI.dll`
- 若检测失败，向导会弹出提示，可选择终止或继续安装
- 一般安装了对应版本 Revit，API 依赖会随 Revit 一起存在

### 部署路径
- DLL: `C:\ProgramData\Autodesk\Revit\Addins\2026\`
- .addin: `%APPDATA%\Autodesk\Revit\Addins\2026\`

## 📝 使用说明

### 打开插件
1. 在 Revit 中切换到平面视图
2. 点击 Ribbon 面板 "绒姆徐特" → "房间管理器"

### 从 DWG 识别
1. 确保已链接 DWG 图纸
2. 点击 "📐 从DWG识别"
3. 选择图纸和识别范围
4. 预览并确认

### 字段管理
1. 点击 "⚙️ 字段管理"
2. 添加/删除/隐藏字段
3. 调整字段顺序
