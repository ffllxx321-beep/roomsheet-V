## 📐 DWG 识别工作流

### 方式一：当前楼层
1. 用户在平面视图中打开插件
2. 选择链接的 DWG 图纸
3. 选择"当前楼层"模式
4. 插件调用 Revit API `NewRooms2()` 自动放置房间
5. 提取 DWG 文字并匹配到房间
6. 自动填充房间名称

### 方式二：框选区域
1. 用户在平面视图中打开插件
2. 选择链接的 DWG 图纸
3. 选择"框选区域"模式
4. 用户在视图中框选识别范围
5. 插件识别框选区域内的闭合区域
6. 创建房间并填充名称

## 📝 使用说明

### 打开插件
1. 在 Revit 中切换到平面视图
2. 点击 Ribbon 面板 "绒姆徐特" → "房间管理器"

### 从 DWG 识别
1. 确保已链接 DWG 图纸
2. 点击 "📐 从DWG识别"
3. 选择图纸和识别范围
4. 预览并确认

### 字段管理
1. 点击 "⚙️ 字段管理"
2. 添加/删除/隐藏字段
3. 调整字段顺序
