## 错误列表与修复方案

### 1. AutoSaveService.Stop() 方法不存在
**错误**: CS1061 - "AutoSaveService"未包含"Stop"的定义  
**位置**: MainViewModel.cs:461  
**原因**: AutoSaveService 类中方法名为 `StopAutoSave()`，不是 `Stop()`  
**修复**: 
```csharp
// 修改前
_autoSaveService?.Stop();

// 修改后
_autoSaveService?.StopAutoSave();
```

---

### 2. DwgRecognitionWizard 构造函数缺少 View 参数
**错误**: CS7036 - 未提供与"DwgRecognitionWizard.DwgRecognitionWizard(Document, View)"的所需参数"view"对应的参数  
**位置**: MainWindow.xaml.cs:141  
**原因**: 构造函数需要 `(Document, View)` 两个参数，但只传了 `Document`  
**修复**:
```csharp
// 修改前
var wizard = new DwgRecognitionWizard(_document!);

// 修改后
var wizard = new DwgRecognitionWizard(_document, _activeView);
```

---

### 3. ExcelService.ReadFromExcel() 方法不存在
**错误**: CS1061 - "ExcelService"未包含"ReadFromExcel"的定义  
**位置**: MainWindow.xaml.cs:159  
**原因**: ExcelService 类中方法名为 `ImportFromExcel()`，不是 `ReadFromExcel()`  
**修复**:
```csharp
// 修改前
var data = excelService.ReadFromExcel(dialog.FileName);

// 修改后
var data = excelService.ImportFromExcel(dialog.FileName);
```

---

### 4. PdfReportService.ExportToPdf() 方法不存在
**错误**: CS1061 - "PdfReportService"未包含"ExportToPdf"的定义  
**位置**: MainWindow.xaml.cs:206  
**原因**: PdfReportService 类中方法名为 `GenerateRoomReport()`，不是 `ExportToPdf()`  
**修复**:
```csharp
// 修改前
pdfService.ExportToPdf(_viewModel.Rooms.ToList(), dialog.FileName);

// 修改后
pdfService.GenerateRoomReport(_viewModel.Rooms.ToList(), dialog.FileName);
```

---

### 5. ValidationWindow 构造函数参数类型错误
**错误**: CS1503 - 参数 1: 无法从"List<RoomData>"转换为"ValidationReport"  
**位置**: MainWindow.xaml.cs:218  
**原因**: ValidationWindow 需要 `ValidationReport` 对象，但传了 `List<RoomData>`  
**修复**:
```csharp
// 修改前
var window = new ValidationWindow(_viewModel.Rooms.ToList());

// 修改后
var validationService = new ValidationService();
var report = validationService.ValidateRooms(_viewModel.Rooms);
var window = new ValidationWindow(report);
```

---

### 6. MainViewModel.SelectedRooms 属性不存在
**错误**: CS1061 - "MainViewModel"未包含"SelectedRooms"的定义  
**位置**: MainWindow.xaml.cs:225  
**原因**: MainViewModel 只有 `SelectedRoom`（单个房间），没有 `SelectedRooms`（集合）  
**修复**:
```csharp
// 修改前
var window = new BatchOperationWindow(_document!, _viewModel.SelectedRooms.ToList());

// 修改后
var selectedRooms = _viewModel.SelectedRoom != null 
    ? new List<RoomData> { _viewModel.SelectedRoom }
    : _viewModel.Rooms.ToList();
var window = new BatchOperationWindow(_document, selectedRooms);
```

---

### 7. FieldManagerWindow 构造函数参数类型错误
**错误**: CS1503 - 参数 1: 无法从"FieldManager"转换为"MainViewModel"  
**位置**: MainWindow.xaml.cs:232  
**原因**: FieldManagerWindow 需要 `MainViewModel`，但传了 `FieldManager`  
**修复**:
```csharp
// 修改前
var window = new FieldManagerWindow(_viewModel.FieldManager);

// 修改后
var window = new FieldManagerWindow(_viewModel);
```

## 核心教训

1. **调用方法前必须检查方法名** - 不要凭直觉猜测，要查看实际定义
2. **构造函数参数必须匹配** - 查看类定义，确保参数类型和数量正确
3. **属性不存在时要检查替代方案** - `SelectedRoom` vs `SelectedRooms`
4. **类型转换要合理** - `List<RoomData>` → `ValidationReport` 需要先调用服务转换
